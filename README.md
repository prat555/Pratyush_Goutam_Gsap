# Pratyush Goutam - Personal Portfolio

A minimal, high-performance personal portfolio website built to showcase my projects, skills, and experience as a Full Stack Developer.

## Features
- **Modern UI/UX:** Clean, dark-themed design with a focus on typography and spacing.
- **Animations:** Smooth scrolling and reveal animations powered by GSAP.
- **Interactive Background:** Custom particle system using Three.js.
- **Responsive:** Fully responsive across all device sizes.
- **No Build Step:** Pure HTML, CSS, and Vanilla JavaScript for maximum performance and easy deployment.

## Tech Stack
- HTML5
- CSS3 (Variables, Grid, Flexbox)
- JavaScript (Vanilla)
- [GSAP](https://gsap.com/) (Animations & ScrollTrigger)
- [Three.js](https://threejs.org/) (Canvas particle background)

## Local Development
Since this is a static website with no build step, you can run it easily:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
*(Optional: Use an extension like Live Server in VS Code for hot-reloading during development).*
